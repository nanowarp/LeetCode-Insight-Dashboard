# 🚀 LeetTrack

LeetTrack is an advanced LeetCode profile analyzer that helps users understand coding performance, strengths, weaknesses, interview readiness, and recent submissions.

## ✨ Features

✅ Analyze LeetCode username instantly  
✅ Total Solved / Easy / Medium / Hard Stats  
✅ Strengths & Weak Areas Detection  
✅ Job Readiness Meter  
✅ Integrity Score  
✅ Recent Accepted Submissions  
✅ 45-Min Mock Interview Mode  
✅ Beautiful Modern UI

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript
- LeetCode Public APIs
